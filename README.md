# Ministry_Roads_Analytics

This project demonstrates **data analytics for public sector road infrastructure projects** in Kenya.  

**Objectives:**  
- Monitor **road project budgets, timelines, and completion rates**  
- Track **contractor performance and compliance**  
- Analyze **regional distribution of projects**  
- Provide insights for **government planning and resource allocation**  

**Skills demonstrated:** SQL, Budget & Expense Analysis, Project Tracking, Contractor Performance Analytics, Dashboard-Ready Queries  

**Projects included (10):**  
- Road projects across major Kenyan counties (e.g., Nairobi, Kisumu, Mombasa, Eldoret, Nakuru, etc.)  
- Each project includes **project name, county, contractor, budget, start/end dates, and status**
